# NovaBrew Coffee Taste Profile Quiz

A personality-based coffee matching quiz built for NovaBrew Coffee Co. The app helps subscribers discover their coffee personality, see a percentage-based flavor profile, and get matched with coffees that fit their taste.

## What It Does

- Guides users through a 7-question lifestyle and abstract personality quiz
- Scores six coffee personalities:
  - Bold Explorer
  - Smooth Operator
  - Cozy Classic
  - Wild Card
  - Bright Idealist
  - Craft Purist
- Shows a percentage breakdown across all personalities
- Recommends a personalized NovaBrew coffee lineup
- Includes result artwork, prompt-matched imagery, and a save-to-clipboard CTA

## Why It Exists

This project was built as a customer personalization prototype for NovaBrew Coffee Co. The business goal is to improve subscriber retention by replacing a generic onboarding experience with a taste-profile moment that feels personal, memorable, and shareable.

## Built With

- Next.js
- React
- Tailwind CSS
